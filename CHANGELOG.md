# Changelog

## 0.4.0 - 2026-07-16

- “选择器件”按钮现在暂时隐藏去耦喵并打开嘉立创 EDA 底部原生器件库，点中不同器件后自动返回并回填；
- 提供独立的“用当前”按钮，明确采用原生库当前已高亮器件；
- 将生成窗口收紧为 1080 × 720，并重做为紧凑、现代的工具面板布局。

## 0.3.0 - 2026-07-16

- 将逐电源域放置改为全局“添加电容”：人工落下一个锚点后，一次生成全部已选择电源网络；
- 各网络按独立行排列，使用各自电源母线和单个 GND 标志，整批失败统一回滚；
- 每个电源域增加主电容小勾选，可快速排除主电容且不丢失原有容值配置。

## 0.2.2 - 2026-07-16

- 将完整 GND 母线及全部支路一次创建为单个稳定导线图元；
- 删除单个已生成电容时按剩余电容坐标重建公共母线，避免留下悬空支路或误删整网。

## 0.2.1 - 2026-07-16

- 阵列内部导线不再逐段显式写入网络名，改由唯一电源/GND 标志通过真实连接自然传播网络；
- 消除每根电容支路旁重复显示的 `GND` 文本。

## 0.2.0 - 2026-07-16

- 将逐电容 GND 短线和重复 GND 标志改为连续公共 GND 母线，每个电源域只保留一个 GND 标志；
- 使用真实电源/GND 导线形成完整并联连接，并兼容两类器件符号的不同引脚间距；
- 接入嘉立创 EDA 原生底部器件库选择，主电容与引脚去耦电容可分别选择和缓存器件。

## 0.1.15 - 2026-07-16

- 按嘉立创 EDA 原理图坐标系修正竖直电容的上下端判定：较大 Y 为屏幕上方的电源端；
- 电源母线放在电容上方，GND 短线与标识放在电容下方，匹配参考电源页样式。

## 0.1.14 - 2026-07-16

- 在 `draw_end` 后重新对比放置前图元 ID 基线，以差集中的稳定图元作为锚点；
- 兼容客户端在结束重复放置工具时替换临时图元 ID 的行为。

## 0.1.13 - 2026-07-16

- `draw_end` 结束放置工具后，按 primitive ID 重新读取已落下的稳定锚点对象；
- 避免使用工具结束前的失效图元包装对象调用 `modify()`。

## 0.1.12 - 2026-07-16

- 捕获用户点下的锚点后调用客户端 `draw_end` 命令，结束官方接口默认保持的重复放置工具；
- 只保留已落下的锚点，然后再执行旋转、容值写入和连接式阵列生成。

## 0.1.11 - 2026-07-15

- 在电容修改、其余器件创建和真实导线创建完成前保持画布命令上下文，避免过早恢复 iframe 导致 `modify()` 抛出 `cmdKey` 异常。

## 0.1.10 - 2026-07-15

- 在恢复扩展窗口前先清理鼠标跟随提示；
- 将已脱离画布上下文时可能抛出 `cmdKey` 的提示清理操作变为幂等操作，不中断电路生成事务。

## 0.1.9 - 2026-07-15

- 同步为父编辑器窗口提供经 iframe 偏移换算的 `clientX/clientY`，兼容官方鼠标放置方法在落点阶段读取父窗口 `window.event` 的实现。

## 0.1.8 - 2026-07-15

- 将 iframe 内的可信点击事件临时桥接到官方放置接口依赖的旧式 `window.event`，解决 Electron 中 `clientX` 解构异常；
- 官方放置调用启动后立即恢复全局事件属性，不污染后续 UI 交互。

## 0.1.7 - 2026-07-15

- 在用户点击处理器的同步阶段立即调用 `placeComponentWithMouse()`，保留 V3.2.149 依赖的真实 `window.event` 和鼠标坐标；
- 预先缓存图元 ID 基线，避免在启动放置工具前异步等待。

## 0.1.6 - 2026-07-15

- 隔离 V3.2.149 在已成功恢复 iframe 后仍抛出的内部 `cmdKey` 异常，不让它中断后续连接式电容阵列生成。

## 0.1.5 - 2026-07-15

- 落点捕获后使用扩展窗口重获焦点，避免 V3.2.149 在放置工具中调用 `activateDocument()` 时抛出 `cmdKey` 异常。

## 0.1.4 - 2026-07-15

- 区分官方接口的临时鼠标图元与用户真正点下的锚点图元；
- 改用新增图元 ID 轮询捕获落点，兼容 V3.2.149 未回调 `SCH_Event.create` 的情况。

## 0.1.3 - 2026-07-15

- 改为按电源域逐个调用官方 `placeComponentWithMouse()` 人工落位；
- 电容阵列使用水平电源母线和逐电容 GND 短导线真实连接；
- 删除自动散放以及每个电容重复放置电源标签的布局；
- Manifest 分开记录器件与导线，支持精确回滚和删除。

## 0.1.2 - 2026-07-15

- 修复 iframe 相对资源路径和 ES module 脚本在客户端未加载的问题；
- 扩展 VIN_SRAM、VDDIN、VIOIN_18 等实际雷达芯片电源脚识别；
- 补充“显示在顶部菜单”的安装配置说明。

## 0.1.1 - 2026-07-15

- 修复器件尚未进入网表时生成窗口无法打开的问题；
- 增加编译后入口运行时回归测试，覆盖无网表器件场景；
- 明确网表只用于补充标签建议，不再成为生成器启动条件。

## 0.1.0 - 2026-07-15

- 建立“去耦喵”嘉立创 EDA V3 扩展；
- 支持选中一个芯片并读取全部引脚及 `.enet` 网络名；
- 支持 POWER 类型、常见名称和已有网络名的多电源域建议；
- 支持手动选择任意引脚、新建/删除电源域和重新分配引脚；
- 支持预设或自定义电源标签；
- 支持每域任意增删主电容和逐引脚去耦电容；
- 支持搜索并缓存用户选择的两引脚电容库器件；
- 支持事务式创建与失败回滚；
- 支持删除单个电容、删除电源域和撤销整批生成；
- 增加 Apache-2.0 许可证、测试和 CI。
